<div style="text-align: justify"> 
In this workshop I will present the basic concepts of LCA and its application with the
<a href="https://github.com/Marcosjnez/latent">latent</a>
R package. LCA is a Categorical Latent Variable Model, where we identify "unobserved" groups in function of response patterns from a set of indicators. As an exploratory method, we test multiple solutions and evaluate the statistical and theoretical relevance to select the "best" solution. We will see how to run LCA with only categorical, only continuous, and combination of item types. We will then discuss how to choose the "best" number of classes, model fit evaluation, and result interpretation. Lastly, we will discuss how to do regressions on the latent classes with the 2-step approach. 
Please follow the latent installation instructions from GitHub before the workshop 
<a href="https://github.com/Marcosjnez/latent">https://github.com/Marcosjnez/latent</a>.
</div>
